About This System

This Emergency Room Triage Prediction system is built as a Final Year Project using machine learning. It predicts the priority level of a patient based on their vital signs and health history, enabling faster and more accurate clinical decisions in emergency settings.

The model is trained on synthetic_medical_triage.csv — verified to have no null values — using an 80/20 train-test split and 10-fold cross-validation for robust and reliable accuracy.
